##  📁 &nbsp; Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/yangvaz/uz-teste

    # Entrar no diretório
    $ cd uz-teste

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn start
```
